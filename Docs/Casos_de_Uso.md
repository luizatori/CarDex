# Casos de Uso e Fluxos do Sistema

## 1. Casos de Uso

| ID | Caso de Uso | Ator | Descrição |
| :--- | :--- | :--- | :--- |
| **UC01** | Visualizar Coleção | Usuário | Ver a grade de cards (preenchidos + vazio) |
| **UC02** | Criar Novo Card | Usuário | Preencher foto, nome, descrição e moldura |
| **UC03** | Editar Card Existente | Usuário | Alterar nome, descrição ou moldura |
| **UC04** | Excluir Card | Usuário | Remover card com confirmação |
| **UC05** | Filtrar Coleção | Usuário | Filtrar por data ou favoritos |
| **UC06** | Selecionar Moldura | Usuário | Escolher estilo visual do card |
| **UC07** | Desbloquear Moldura | Sistema | Liberar moldura ao atingir X carros |
| **UC08** | Personalizar Perfil | Usuário | Editar nome, texto e visibilidade dos widgets |
| **UC09** | Gerenciar Favoritos | Usuário | Selecionar até 4 carros favoritos |
| **UC10** | Alternar Tema Claro/Escuro | Usuário | Trocar entre modo claro e escuro |
| **UC11** | Navegar entre Telas | Usuário | Alternar entre Home e Perfil via BottomNav |
